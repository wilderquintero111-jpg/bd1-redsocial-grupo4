## Descripción del caso

La red social Pascualina es una propuesta de sistema informático que promueve la comunicación, la interacción y la colaboración de los estudiantes de la institución. En la actualidad, existen plataformas institucionales, pero son principalmente formales y, por lo tanto, no facilitan la interacción social, la transferencia de conocimientos entre pares (práctica habitual en las conversaciones en redes sociales), la organización de actividades académicas o de actividades extracurriculares de forma dinámica.

Este proyecto de investigación persigue el propósito de diseñar un modelo conceptual de base de datos que soporte el funcionamiento de una red social estudiantil, a través del modelo Entidad-Relación (MER). La plataforma dará la oportunidad a los estudiantes de crear vidas y perfiles de perfil, donde podrán introducir sus datos académicos, intereses y habilidades, conectando así a estudiantes con intereses parecidos o estudiantes con mayor experiencia que pueden ayudar a ofrecer o recibir apoyo académico o mentoría. 

A su vez, el sistema permitirá crear publicaciones de información relevante como preguntas académicas, recursos, noticias o contenidos en relación con la vida universitaria. Se podrán gestionar y crear grupos de estudio, contribuyendo así a la colaboración de los estudiantes en torno a materias, proyectos, intereses comunes,
Para soportar estas funcionalidades, se diseñó un modelo conceptual de base de datos que identifica las principales entidades, atributos y relaciones necesarias para garantizar una estructura organizada, coherente y eficiente de la información. Este modelo constituye la base para el posterior desarrollo lógico y físico del sistema.

Este proyecto permite aplicar los conceptos fundamentales del diseño de bases de datos, especialmente el modelo Entidad-Relación, contribuyendo al desarrollo de competencias en análisis, modelado y estructuración de sistemas de información.
